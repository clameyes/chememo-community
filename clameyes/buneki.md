分液操作は有機化学実験において最も基本的かつ重要な操作の一つです。この記事では、分液操作を成功させるためのコツを図解付きで詳しく解説します。

## 分液操作とは

分液操作は、**水と混じり合わない2つの液体を分離する技術**です。主に以下の目的で使用されます：

- 反応混合物から生成物を抽出
- 不要な水溶性不純物を除去
- pH調整による精製

---

## 分液漏斗の選び方と準備

### サイズの選び方

| 処理液量 | 推奨漏斗サイズ |
|---------|--------------|
| 〜50 mL | 100 mL |
| 50〜100 mL | 200 mL |
| 100〜200 mL | 500 mL |
| 200〜500 mL | 1 L |

**ポイント**: 処理する液量の**2〜3倍の容量**を選択

### コックの種類と特徴

| 種類 | メリット | デメリット |
|-----|---------|----------|
| ガラスコック | 有機溶媒に強い | 固着しやすい、グリス必要 |
| テフロンコック | 固着しにくい、グリス不要 | やや高価 |

## 分液漏斗の正しい持ち方

```
【正しい持ち方】

        栓を押さえる
            ↓
        ┌───○───┐
        │       │
        │～～～～│ ← 有機層
        │＝＝＝＝│ ← 境界面
        │■■■■│ ← 水層
        └───┬───┘
            │
       ┌────┴────┐
       │  コック  │ ← 反対の手で操作
       └────┬────┘
            │
            ▼
```

**重要ポイント**:
1. 上の栓は必ず手で押さえる（内圧で飛ぶ危険）
2. コックは反対の手で操作
3. 漏斗本体は持たない

## 振とうの正しいテクニック

### Step 1: セットアップ
```
     ＼栓を押さえる
      ＼↓
   ○───────○
   │～～～～～│  横に傾ける
   │＝＝＝＝＝│  コックを上向きに
   │■■■■■│
   ○───────○
         ↗
    コックを上に
```

### Step 2: ガス抜き
```
   コックを開く
       ↓
   ○───────◎ ← 開
   │     ／│
   │ ガス→ │  「シュッ」という音
   │     ↗│
   ○───────○

   ※ 顔や人に向けないこと！
```

### Step 3: 振とう
```
   ┌───────┐     ┌───────┐
   │ ～～～ │ ←→ │ ～～～ │
   │ ＝＝＝ │     │ ＝＝＝ │
   │ ■■■ │     │ ■■■ │
   └───────┘     └───────┘

   穏やかに3〜5秒振る
```

### Step 4: 繰り返し

**ガス抜き → 振とう** を **3〜5回** 繰り返す

## 有機層・水層の見分け方チートシート

### 判別フローチャート

```
溶媒の密度は水より大きい？
        │
    ┌───┴───┐
    ↓       ↓
   YES      NO
    │       │
    ↓       ↓
 下が有機層  上が有機層
 上が水層   下が水層
```

### 代表的な溶媒の密度一覧

| 溶媒 | 密度 (g/mL) | 水との位置関係 |
|-----|------------|--------------|
| ジクロロメタン | 1.33 | **下** |
| クロロホルム | 1.49 | **下** |
| 四塩化炭素 | 1.59 | **下** |
| 酢酸エチル | 0.90 | 上 |
| ジエチルエーテル | 0.71 | 上 |
| トルエン | 0.87 | 上 |
| ヘキサン | 0.66 | 上 |

### 迷ったときの確認方法

**水滴テスト**: 分離後の各層に水を1滴加える
- 水滴が混ざる → 水層
- 水滴が沈むor浮く → 有機層

## エマルジョン対処フローチャート

エマルジョン（乳化）が発生したら、以下のフローに従って対処：

```
┌─────────────────────────────────┐
│    エマルジョン発生！           │
└─────────────┬───────────────────┘
              ↓
┌─────────────────────────────────┐
│ Step 1: 静置する（30分〜1時間）  │
│         温めると効果的           │
└─────────────┬───────────────────┘
              ↓
         分離した？
        ┌───┴───┐
        YES     NO
        │       │
        ↓       ↓
      完了    次へ
              │
              ↓
┌─────────────────────────────────┐
│ Step 2: 飽和食塩水を加える       │
│         (全量の10〜20%)          │
└─────────────┬───────────────────┘
              ↓
         分離した？
        ┌───┴───┐
        YES     NO
        │       │
        ↓       ↓
      完了    次へ
              │
              ↓
┌─────────────────────────────────┐
│ Step 3: セライトろ過            │
│         少量のセライトを加えろ過  │
└─────────────┬───────────────────┘
              ↓
         分離した？
        ┌───┴───┐
        YES     NO
        │       │
        ↓       ↓
      完了    次へ
              │
              ↓
┌─────────────────────────────────┐
│ Step 4: 遠心分離（最終手段）     │
│         3000 rpm, 5〜10分        │
└─────────────────────────────────┘
```

### エマルジョンを防ぐコツ

1. **振りすぎない**: 穏やかに振る
2. **界面活性剤を避ける**: 石鹸で洗った器具をよくすすぐ
3. **濃度を薄める**: 濃い溶液は乳化しやすい

## よくあるトラブルと対策

### 問題1: コックから液漏れ

| 原因 | 対策 |
|-----|------|
| コックの緩み | しっかり閉める |
| グリス不足 | 適量のグリスを塗る |
| ゴミの付着 | コックを清掃する |

### 問題2: 栓が飛ぶ

| 原因 | 対策 |
|-----|------|
| ガス抜き不足 | こまめにガス抜き |
| 反応性ガス発生 | 少量ずつ加える |
| 栓を押さえていない | 必ず手で押さえる |

### 問題3: 分離が不完全

| 原因 | 対策 |
|-----|------|
| 静置時間不足 | 十分に待つ |
| 溶媒の相溶性 | 塩を加える |
| 温度が低い | 室温で操作 |

---

## 実験のコツまとめ

1. **大は小を兼ねない**: 適切なサイズの漏斗を選ぶ
2. **ガス抜きは命**: 必ず毎回行う
3. **穏やかに**: 振りすぎはエマルジョンの元
4. **迷ったら水滴テスト**: 有機層・水層の確認に
5. **両層を捨てない**: 実験が終わるまで保管
