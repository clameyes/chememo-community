光延反応（Mitsunobu反応）は、アルコールを求核剤で置換する強力な手法です。この記事では、反応条件の選択から精製のコツまで詳しく解説します。

## 光延反応とは

光延反応は**アルコールのOH基を脱離基に変換し、求核置換を行う反応**です。

```
【基本反応式】

R-OH + HNu + PPh₃ + DEAD → R-Nu + Ph₃P=O + DEAD-H₂

  アルコール  求核剤  トリフェニル  DEAD    生成物   副生成物
              (pKa<15) ホスフィン
```

### 特徴

- **立体反転**: Walden反転により立体が反転
- **温和な条件**: 室温〜0℃で進行
- **広い適用範囲**: 1級・2級アルコールに使用可能

---

## 反応メカニズム

```
【光延反応のメカニズム】

Step 1: ベタイン形成
─────────────────────────────
PPh₃ + DEAD → [Ph₃P⁺─N⁻─N─CO₂Et]
                  │
              ベタイン中間体

Step 2: アルコールの活性化
─────────────────────────────
      [Ph₃P⁺─N⁻...]
            ↓
R─OH + ベタイン → R─O─PPh₃⁺ + DEAD-H⁻
                    │
              アルコキシホスホニウム

Step 3: 求核置換（SN2）
─────────────────────────────
         Nu⁻
          ↓
    R─O─PPh₃⁺ → R─Nu + Ph₃P=O
          ↑
     背面攻撃（立体反転）
```

## 試薬の選択

### アゾジカルボン酸エステル

| 試薬 | 略称 | 特徴 | 用途 |
|-----|------|------|------|
| DEAD | - | 最も一般的 | 汎用 |
| DIAD | - | DEADより安定、除去しやすい | 汎用 |
| ADDP | - | 水溶性副生成物 | 精製容易 |
| TMAD | - | 副生成物が結晶化 | 精製容易 |

### ホスフィン

| 試薬 | 特徴 | 用途 |
|-----|------|------|
| PPh₃ | 最も一般的、安価 | 汎用 |
| PBu₃ | より反応性高い | 不活性基質 |
| P(o-Tol)₃ | 立体障害あり | 選択的反応 |

### 求核剤（HNu）

| 種類 | 例 | pKa目安 |
|-----|-----|--------|
| カルボン酸 | PhCO₂H, AcOH | ~5 |
| イミド | フタルイミド | ~8 |
| フェノール | PhOH | ~10 |
| アジド | HN₃ (DPPA使用) | ~5 |
| チオール | PhSH | ~7 |

**注意**: pKa > 13の求核剤は反応しにくい

```
┌─────────────────────────────────┐
│ Step 3: 試薬の当量決定          │

└─────────────────────────────────┘
```

## 標準的な反応手順

### 試薬添加の順序（重要）

```
【推奨する添加順序】

1. アルコール + 求核剤（溶媒中）
         ↓
2. PPh₃を加える
         ↓
3. 0℃に冷却
         ↓
4. DEADをゆっくり滴下
         ↓
5. 室温で撹拌

※ DEAD/DIADは最後に滴下！
```

### 代表的なレシピ

```
【標準条件】

アルコール (1.0 eq)
求核剤     (1.2 eq)
PPh₃       (1.3 eq)
DEAD       (1.3 eq)
THF        (0.2-0.5 M)
0℃ → rt, 1-12 h
```

### 溶媒の選択

| 溶媒 | 特徴 |
|-----|------|
| THF | 最も一般的 |
| トルエン | 高温反応に |
| DCM | 低温反応に |
| DMF | 極性基質に |

## 精製のコツ

### 副生成物の問題

光延反応の最大の課題は**Ph₃P=Oとヒドラジン誘導体の除去**です。

```
【主な副生成物】

Ph₃P=O (トリフェニルホスフィンオキシド)
- Rf値が生成物と近いことが多い
- カラムで除去困難な場合あり

DEAD-H₂ (ヒドラジン誘導体)
- 極性が高い
- 通常はカラムで除去可能
```

### 精製戦略フローチャート

```
┌─────────────────────────────────┐
│ 反応完了後                      │
└─────────────┬───────────────────┘
              ↓
┌─────────────────────────────────┐
│ Method 1: 直接カラム            │
│ - ヘキサン/酢エチ系             │
│ - Ph₃P=Oが分離できるか確認      │
└─────────────┬───────────────────┘
              ↓
      Ph₃P=Oと分離できる？
        ┌───┴───┐
        YES     NO
        │       │
        ↓       ↓
      完了    次へ
              │
              ↓
┌─────────────────────────────────┐
│ Method 2: 沈殿除去              │
│ - ヘキサンを大量に加える        │
│ - Ph₃P=Oが沈殿                  │
│ - ろ過後にカラム                │
└─────────────┬───────────────────┘
              ↓
      分離できた？
        ┌───┴───┐
        YES     NO
        │       │
        ↓       ↓
      完了    次へ
              │
              ↓
┌─────────────────────────────────┐
│ Method 3: 再結晶・蒸留          │
│ - 生成物が結晶性なら再結晶      │
│ - 低分子なら蒸留                │
└─────────────────────────────────┘
```

### Ph₃P=O除去のテクニック

| 方法 | 条件 | 適用 |
|-----|------|------|
| ヘキサン沈殿 | 大量ヘキサン添加 | 汎用 |
| シリカゲルカラム | 極性低めで溶出 | Rf差がある場合 |
| 塩基性アルミナ | 中性〜塩基性条件 | 酸に敏感な場合 |
| 洗浄 | 5% HCl水溶液 | 塩基性生成物 |

## トラブルシューティング

### 反応が進行しない

| 原因 | 対策 |
|-----|------|
| 求核剤のpKaが高すぎる | より酸性の求核剤に変更 |
| 立体障害 | PBu₃に変更、温度上昇 |
| 試薬が劣化 | 新しい試薬を使用 |
| 溶媒が湿っている | 脱水溶媒を使用 |

### 収率が低い

| 原因 | 対策 |
|-----|------|
| 試薬不足 | 当量を増やす |
| 副反応 | 低温で反応 |
| 精製ロス | 精製法を変更 |

### 立体選択性が低い

| 原因 | 対策 |
|-----|------|
| E1競合（3級） | 2級以下のアルコールを使用 |
| 基質のラセミ化 | 低温、短時間 |

### 副反応が起こる

```
【よくある副反応】

1. エステル交換
   原因: DEADのエステル部位が反応
   対策: DIADに変更

2. 脱水（オレフィン生成）
   原因: E2反応
   対策: 低温、塩基性条件を避ける

3. アジリジン形成（アミノアルコール）
   原因: 分子内置換
   対策: 保護基を使用
```

## 実用的なTips

### 試薬の保存

| 試薬 | 保存条件 | 劣化の兆候 |
|-----|---------|----------|
| DEAD/DIAD | 冷蔵、遮光 | 黄色→無色で劣化 |
| PPh₃ | 室温、乾燥 | 酸化すると効かない |

### スケールアップの注意点

1. **発熱**: DEAD添加は必ずゆっくり
2. **濃度**: 希釈気味のほうが副反応少ない
3. **撹拌**: 効率的な混合が重要

### グリーンケミストリー代替法

| 試薬系 | 利点 |
|-------|------|
| ADDP/PBu₃ | 副生成物が水溶性 |
| CMBP | ワンポット、試薬1種類 |
| 担持試薬 | ろ過で除去可能 |

## 適用例

### エーテル合成

```
R-OH + ArOH → R-O-Ar
（フェノールを求核剤として）
```

### エステル合成

```
R-OH + R'CO₂H → R-O-CO-R'
（カルボン酸を求核剤として）
```

### アジド導入

```
R-OH + DPPA → R-N₃
（DPPAを求核剤として、Zn(N₃)₂も使える）
```

### イミド合成（Gabriel合成の一部）

```
R-OH + フタルイミド → R-フタルイミド
（後でヒドラジン分解でアミンに）
```

## 実験のコツまとめ

1. **添加順序を守る**: DEADは最後に滴下
2. **試薬は新鮮に**: 特にDEADの劣化に注意
3. **pKa < 13**: 求核剤の酸性度を確認
4. **Ph₃P=O対策**: ヘキサン沈殿を試す
5. **低温開始**: 0℃から始めて必要なら昇温

---

## まとめ

光延反応は立体反転を伴うアルコールの変換に非常に有用な反応です。試薬の添加順序と当量、精製法を押さえれば、高収率で目的物を得ることができます。Ph₃P=Oの除去が最大の課題ですが、沈殿法やカラム条件の工夫で対処可能です。
