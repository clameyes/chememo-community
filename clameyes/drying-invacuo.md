減圧乾燥は有機化学実験において最も頻繁に行う操作の一つです。この記事では、エバポレーターとデシケーターの使い分けから、トラブル対策まで詳しく解説します。

## 減圧乾燥の原理

```
【減圧による沸点低下】

気圧(mmHg)   水の沸点   ジクロロメタン沸点
─────────────────────────────────────
   760        100℃         40℃
   400         83℃         22℃
   100         52℃         -7℃
    20         22℃        -30℃

→ 減圧することで低温でも蒸発可能
```

**メリット**:
- 熱に弱い化合物も乾燥できる
- 乾燥時間が短縮
- 高沸点溶媒も除去可能

## エバポレーターの使い方

### 構成と役割

```
【ロータリーエバポレーター】

                  冷却水
                    ↓
              ┌─────────┐
              │ 冷却管  │ ← 溶媒蒸気を凝縮
              └────┬────┘
                   │
      モーター →┌──┴──┐
              │回転 │
              └──┬──┘
                 │
            ┌────┴────┐
            │         │ ← ナス型フラスコ（サンプル）
            │ ～～～  │   斜めに回転
            └─────────┘
                 │
            ─────┴─────
              湯浴
```

### 使用手順

1. **冷却水を流す**: 冷却管を冷やす
2. **湯浴を設定**: 溶媒沸点 + 10-20℃
3. **フラスコを接続**: クリップでしっかり固定
4. **減圧開始**: ゆっくりバルブを開ける
5. **回転開始**: 60-100 rpm程度
6. **乾燥完了後**: 先に回転停止→大気開放→フラスコ取り外し

### 湯浴温度の目安

| 溶媒 | 沸点(℃) | 推奨湯浴温度 |
|-----|---------|------------|
| ジエチルエーテル | 35 | 30-35℃（または氷浴） |
| ジクロロメタン | 40 | 35-40℃ |
| アセトン | 56 | 40-45℃ |
| メタノール | 65 | 45-50℃ |
| 酢酸エチル | 77 | 50-55℃ |
| エタノール | 78 | 50-55℃ |
| 水 | 100 | 60-70℃ |
| トルエン | 111 | 60-70℃ |
| DMF | 153 | 70-80℃ |
| DMSO | 189 | 80-90℃ |

## デシケーター・真空ラインの使い方

### 用途

- エバポレーター後の残存溶媒除去
- 固体サンプルの乾燥
- 結晶の乾燥
- 長時間乾燥が必要な場合

### 真空デシケーターの構成

```
【真空デシケーター】

      コック
        │
    ┌───┴───┐
    │ ○───  │ ← 真空ポンプへ
    │       │
    │ ┌───┐ │
    │ │ S │ │ ← サンプル（ろ紙の上に）
    │ └───┘ │
    │───────│ ← 棚板
    │■■■■■│ ← 乾燥剤
    └───────┘
```

### 乾燥剤の選び方

| 乾燥剤 | 吸水能力 | 特徴 |
|-------|---------|------|
| P₂O₅（五酸化二リン） | 最強 | 強力だが取扱注意 |
| シリカゲル | 中 | 安価、再生可能 |
| CaCl₂（塩化カルシウム） | 中 | アミンには不向き |
| MgSO₄（硫酸マグネシウム） | 中 | 汎用的 |
| Na₂SO₄（硫酸ナトリウム） | 弱 | 水のみ除去 |

## 乾燥時間の目安

### エバポレーター

| 溶媒量 | 溶媒種 | 目安時間 |
|-------|--------|---------|
| 50 mL | ジクロロメタン | 5-10分 |
| 50 mL | 酢酸エチル | 10-15分 |
| 50 mL | メタノール | 15-20分 |
| 50 mL | 水 | 30-60分 |

### 真空ライン/デシケーター

| 状態 | 目安時間 |
|------|---------|
| 湿った結晶 | 2-4時間 |
| 固体（残存溶媒少量） | 1-2時間 |
| 高沸点溶媒（DMF, DMSO） | 一晩以上 |
| オイル状 | 数時間〜一晩 |

## 乾燥完了の確認方法

### 方法1: 重量変化

```
1. 初回計量
2. 30分乾燥
3. 再計量
4. 重量変化なし → 乾燥完了
```

### 方法2: NMR確認

- ¹H NMRで溶媒ピークを確認
- 溶媒ピークが消失 or 無視できるレベル → OK

### 方法3: 真空度チェック

- デシケーターの真空度が維持される → 乾燥完了
- 真空度が落ちる → まだ溶媒が残存

## トラブルシューティング

### 突沸（バンピング）

```
【突沸の様子】

      急に泡立つ
         ↓
    ┌─────────┐
    │ ぶくぶく │
    │ ○○○○○ │
    │ ○○○○○ │
    │ ↑↑↑↑↑ │
    └─────────┘

  → サンプルが飛び散る危険！
```

| 原因 | 対策 |
|-----|------|
| 急激な減圧 | ゆっくりバルブを開ける |
| 過加熱 | 湯浴温度を下げる |
| 撹拌不足 | 回転数を上げる |
| フラスコが小さい | 大きめのフラスコを使用 |

### 凝固（固まる）

| 原因 | 対策 |
|-----|------|
| 融点以下になった | 湯浴温度を上げる |
| 濃縮されすぎた | 溶媒を追加して再開 |
| 水分で固化 | 無水溶媒で溶解しなおす |

### 乾燥しない

| 原因 | 対策 |
|-----|------|
| 真空度不足 | ポンプ、配管を確認 |
| 高沸点溶媒 | 温度を上げる or 長時間乾燥 |
| 水分の混入 | 無水溶媒で共沸 |

### サンプルが減る

| 原因 | 対策 |
|-----|------|
| 昇華した | 低温で乾燥 |
| 揮発性がある | 減圧度を下げる |
| 溶媒と共沸 | 別の溶媒に置換 |

## 高沸点溶媒の除去

### DMF、DMSOの除去テクニック

```
【段階的除去】

1. エバポレーター（高温）
   → 大部分を除去

2. トルエン共沸（3回）
   → さらに除去

3. 真空ライン（一晩）
   → 完全除去
```

### 共沸に使える溶媒

| 高沸点溶媒 | 共沸溶媒 |
|-----------|---------|
| DMF | トルエン |
| DMSO | トルエン |
| 水 | トルエン、エタノール |
| ピリジン | トルエン |

## 安全上の注意

### エバポレーター

1. **ガラス器具の破損**: 減圧前にヒビがないか確認
2. **突沸**: ゆっくり減圧、適切な温度
3. **トラップの凍結**: ドライアイストラップは定期確認

### デシケーター

1. **急激な大気開放**: ゆっくりコックを開ける
2. **P₂O₅の取扱**: 水と激しく反応、皮膚に付けない
3. **真空の維持**: グリスを適切に塗布

---

## 実験のコツまとめ

1. **フラスコは大きめに**: 突沸対策、半分以下が目安
2. **湯浴は溶媒に合わせる**: 高すぎると分解、低すぎると時間がかかる
3. **回転は適度に**: 速すぎると跳ねる、遅すぎると乾燥が遅い
4. **高沸点溶媒は共沸**: トルエンで3回が基本
5. **最終乾燥は真空ライン**: NMR測定前の習慣に

---

## まとめ

減圧乾燥は適切な条件を選べば、迅速かつ安全に溶媒を除去できます。エバポレーターで大部分を除去し、デシケーターや真空ラインで仕上げる二段階が基本です。高沸点溶媒は共沸を活用して効率よく除去しましょう。
