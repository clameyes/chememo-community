TLC（薄層クロマトグラフィー）は反応のモニタリングや精製条件の検討に欠かせない技術です。この記事では、展開溶媒の選び方からRf値の調整まで詳しく解説します。

## TLCの基本原理

```
【TLCの原理】

       展開前          展開後
         │               │
     ┌───┴───┐       ┌───┴───┐
     │       │       │       │
     │       │       │   ●   │ ← 低極性成分（上へ）
     │       │       │       │
     │       │  ───→ │   ●   │ ← 高極性成分（下に）
     │       │       │       │
     │ ● ●  │       │───────│ ← 原点
     └───────┘       └───────┘
       原点         溶媒前線
```

**ポイント**:
- シリカゲル（固定相）は極性が高い
- 極性が高い化合物ほど吸着されやすい（Rf値が小さい）
- 極性が低い化合物ほど移動しやすい（Rf値が大きい）

## 展開溶媒の極性チャート

### 溶媒極性順位（弱→強）

```
【極性の弱い順】

ヘキサン ─→ トルエン ─→ ジクロロメタン ─→ 酢酸エチル ─→ アセトン ─→ メタノール
   │          │            │              │            │          │
  極性        弱            ↓              ↓            ↓          極性
  最弱        ─────────────────────────────────────────────        最強
```

### 溶媒の詳細一覧

| 溶媒 | 略称 | 極性指数 | 特徴 |
|-----|-----|---------|------|
| ヘキサン | Hex | 0.0 | 基本溶媒、非極性 |
| シクロヘキサン | c-Hex | 0.0 | ヘキサンと同等 |
| トルエン | Tol | 2.4 | 芳香族に有効 |
| ジクロロメタン | DCM | 3.1 | 汎用性高い |
| クロロホルム | CHCl₃ | 4.1 | DCMより極性やや高い |
| ジエチルエーテル | Et₂O | 2.8 | 揮発性高い |
| 酢酸エチル | AcOEt | 4.4 | 最も汎用的 |
| アセトン | - | 5.1 | 強極性 |
| アセトニトリル | MeCN | 5.8 | 強極性 |
| メタノール | MeOH | 5.1 | アミン類に有効 |

## 展開溶媒選択フローチャート

```
┌─────────────────────────────────┐
│ Step 1: 化合物の極性を推定      │
│ (官能基から判断)                │
└─────────────┬───────────────────┘
              ↓
┌─────────────────────────────────┐
│ Step 2: 基本条件でTLC           │
│ - 低極性: ヘキサン/酢エチ = 4:1 │
│ - 中極性: ヘキサン/酢エチ = 1:1 │
│ - 高極性: 酢エチ 100% or +MeOH  │
└─────────────┬───────────────────┘
              ↓
        Rf値はいくつ？
       ┌────┴────┐
       │         │
    0.2-0.5   それ以外
       │         │
       ↓         ↓
    最適！    調整必要
              │
      ┌───────┴───────┐
      ↓               ↓
   Rf < 0.2       Rf > 0.5
      │               │
      ↓               ↓
  極性を上げる    極性を下げる
```

## Rf値の読み方

### Rf値の計算

```
       溶媒前線 ───────────────
                    ↑
                    │ b
                    │
         スポット ──●──
                    ↑
                    │ a
                    │
          原点 ───────────────

      Rf = a / b（スポット中心までの距離）
```

### 理想的なRf値

| Rf値 | 評価 | 対策 |
|------|------|------|
| 0.0 - 0.1 | 低すぎる | 極性を上げる |
| 0.1 - 0.2 | やや低い | 少し極性を上げる |
| **0.2 - 0.5** | **最適** | この範囲を目指す |
| 0.5 - 0.7 | やや高い | 少し極性を下げる |
| 0.7 - 1.0 | 高すぎる | 極性を下げる |

## Rf値の調整方法

### Rf値を上げたい場合（スポットが動かない）

```
現在の条件: ヘキサン/酢エチ = 9:1 (Rf = 0.1)

調整方法:
1. 酢酸エチルの割合を増やす
   → 8:2 → 7:3 → 6:4 ...

2. より極性の高い溶媒を少量加える
   → +1% MeOH
```

### Rf値を下げたい場合（スポットが上がりすぎる）

```
現在の条件: ヘキサン/酢エチ = 1:1 (Rf = 0.8)

調整方法:
1. ヘキサンの割合を増やす
   → 6:4 → 7:3 → 8:2 ...

2. より極性の低い溶媒に変更
   → ヘキサン/ジクロロメタン系
```

### 調整の目安表

| 変更 | Rf値への影響 |
|-----|------------|
| 酢エチ比率 +10% | Rf +0.1〜0.15 |
| MeOH 1%追加 | Rf +0.15〜0.2 |
| ヘキサン比率 +10% | Rf -0.1〜0.15 |

```

     UV lamp

    │   ◯   │ ← 蛍光が消える部分 = スポット
```

## よくある失敗と対処法

### 失敗1: スポットが横に広がる（テーリング）

```
  正常          テーリング
    │               │
    ●               ◗
    │               │
```

| 原因 | 対策 |
|-----|------|
| アミンやカルボン酸 | 塩基(トリエチルアミン)or酸(酢酸)を1%添加 |
| スポット量が多い | 薄い溶液を使用 |
| 過飽和 | 新しい展開槽で |

### 失敗2: スポットが縦に伸びる

| 原因 | 対策 |
|-----|------|
| 展開槽が飽和していない | 展開前に十分飽和させる |
| スポット直径が大きい | 細いキャピラリーで小さく |
| 溶媒前線が乱れている | プレートを垂直に立てる |

### 失敗3: スポットが見えない

| 原因 | 対策 |
|-----|------|
| 化合物がUV不活性 | 発色試薬を使用 |
| 濃度が薄い | 濃い溶液をスポット |
| 揮発した | すぐに検出 |

### 失敗4: 全てのスポットが原点に残る

| 原因 | 対策 |
|-----|------|
| 溶媒の極性が低すぎる | 酢エチ比率を上げる |
| 化合物が極性すぎる | MeOHを加える |
| 塩になっている | 酸/塩基でフリー体に |

### 失敗5: 全てのスポットが溶媒前線に達する

| 原因 | 対策 |
|-----|------|
| 溶媒の極性が高すぎる | ヘキサン比率を上げる |
| 化合物が非極性すぎる | ヘキサン/トルエン系に |

## カラムクロマトグラフィーへの応用

TLCで見つけた条件をカラムに応用する際の目安：

```
【TLCからカラムへ】

TLCでのRf値 → カラム溶媒の選択
   0.3       → TLCと同じ溶媒でOK
   0.2       → やや極性を上げる
   0.4-0.5   → やや極性を下げる
```

**経験則**: TLCでRf = 0.2-0.3の条件がカラムに最適

---

## 実験のコツまとめ

1. **まずは定番から**: ヘキサン/酢エチ系で試す
2. **Rf = 0.3を狙う**: この値が分離に最適
3. **テーリングにはアミン**: 1%トリエチルアミン添加
4. **展開槽は飽和させる**: 最低15分は待つ
5. **スポットは小さく**: 細いキャピラリーで

---

## まとめ

TLC展開溶媒の選択は、化合物の極性を見極めることから始まります。この記事のチャートを参考に、適切な溶媒を選び、Rf値を0.2-0.5の範囲に調整してください。カラムクロマトグラフィーの条件検討にも直結する重要なスキルです。
