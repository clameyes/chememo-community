「カラムの溶媒ってどうやって決めるの？」——カラムクロマトグラフィーで最も重要な準備が展開溶媒の選択です。溶媒が適切でなければ、分離がうまくいかないか、大量の溶媒と時間を浪費することになります。この記事では、TLC を使った条件検討から実際のカラム操作で使う溶媒の決め方までを解説します。対象読者は、カラム操作の基本は知っているが溶媒選択に自信がない学部生の方です。カラムの手順全体は「[カラムクロマトグラフィーのやり方](/posts/chememo/column-chromatography)」を参照してください。

## TLC による条件検討

カラムの展開溶媒は、事前に **TLC（薄層クロマトグラフィー）** で条件を絞り込んでから決めます。カラムと TLC は同じシリカゲル吸着の原理で分離するため、TLC の結果はそのままカラムの条件に応用できます。

検討の手順は以下のとおりです。

1. TLC プレートに試料（反応混合物）をスポットする
2. まず低極性の溶媒（ヘキサン 100 % など）で展開し、Rf 値を確認する
3. 極性を段階的に上げて（酢酸エチルの割合を増やすなど）、**目的物の Rf 値が 0.2〜0.4** になる溶媒比率を探す
4. その溶媒比率で目的物と不純物のスポットが十分に離れているか確認する

**Rf 値 0.2〜0.4 が最適な範囲**です。Rf が小さすぎると（0.1 以下）溶出に膨大な溶媒が必要になり、大きすぎると（0.5 以上）バンドが広がって分離が不十分になります。

実験者の立場からの補足ですが、TLC は 1 回の展開に数分しかかかりません。溶媒比率を 3〜4 パターン試しても 20 分程度です。この手間を惜しんでいきなりカラムを流すと、何時間もかけた末に分離失敗ということになりかねません。TLC の段階で十分に条件を詰めることが、結果的に最短ルートです。

## 溶媒の極性と溶出順序

シリカゲルカラムでは、**極性の低い化合物が先に溶出し、極性の高い化合物が後から出てくる**のが基本です。これはシリカゲル表面のシラノール基（$\ce{Si-OH}$）が極性の高い化合物を強く吸着するためです。

展開溶媒の極性を上げると、溶媒がシリカゲルへの吸着で化合物と競合するため、化合物の溶出が速くなります。つまり、溶媒の極性を変えることで「化合物をどのタイミングで出すか」をコントロールできます。

溶出順序の予測には TLC の Rf 値が役立ちます。Rf 値が大きいスポットほど先にカラムから溶出します。また、CV（Column Volume）= 1 / Rf という近似式がよく使われます[^1]。たとえば Rf = 0.25 の化合物は約 4 CV（カラム体積の 4 倍量の溶媒）で溶出すると予測できます。

ただし、この式はあくまで目安です。実際にはカラムの充填状態や流速によって前後するため、フラクションの TLC 確認は必ず行ってください。

## よく使われる展開溶媒

有機化学の研究室で最もよく使われる展開溶媒を極性の順に並べます。

| 溶媒 | 極性 | 特徴 |
|-----|-----|------|
| ヘキサン（$\ce{C6H14}$） | 最低 | 非極性の基準溶媒 |
| トルエン | 低 | 芳香族化合物の分離に |
| ジクロロメタン（$\ce{CH2Cl2}$） | 中 | 極性化合物にも対応 |
| 酢酸エチル（$\ce{AcOEt}$） | 中〜高 | 最も汎用性が高い |
| アセトン | 高 | 極性の高い化合物に |
| メタノール（$\ce{MeOH}$） | 最高 | 強極性化合物の溶出に |

実際のカラム操作では**ヘキサン / 酢酸エチル**の混合比を変える系が最もよく使われます。この組み合わせだけで、非極性から中程度の極性を持つ化合物の大半に対応できます。

極性の高い化合物（アミン、カルボン酸、アルコールなど）の分離にはジクロロメタン / メタノール系が適しています。ただし、メタノールの割合が高すぎるとシリカゲルが溶出してしまうため、メタノールは **10 % 以下** に抑えるのが一般的です。

**安全上の注意**: カラム操作では大量の有機溶媒を使います。ヘキサンと酢酸エチルはいずれも引火性が高いため、ドラフトチャンバー内または十分な換気のある場所で作業してください。使用後の溶媒は有機廃液容器に分別廃棄します。

## グラジエント溶出

目的物と不純物の極性差が大きい場合は、展開溶媒の極性を途中で段階的に上げる**グラジエント溶出**が有効です。

たとえば、低極性の不純物が多い反応混合物を精製する場合、次のように溶媒を切り替えます。

1. ヘキサン / 酢酸エチル = 9:1 で低極性の不純物を先に流す
2. 不純物が出きったことを TLC で確認する
3. 比率を 7:3 に切り替えて目的物を溶出させる

グラジエント溶出を使うメリットは、**最初から高極性の溶媒で流す場合に比べて分離が改善される**点です。低極性の不純物が目的物と一緒に出てしまうのを防げます。

溶媒比率の切り替えタイミングは TLC で判断します。フラクションの TLC を確認し、不要なスポットが出なくなったところで極性を上げるのが確実です。

実験者の立場からの補足ですが、溶媒比率を急激に変えると、カラム内で溶媒の境界面が生じてバンドが乱れることがあります。比率は一度に大きく変えず、9:1 → 8:2 → 7:3 のように段階的に変える方が安全です。

## 溶媒調製のポイント

カラム操作を始める前に、展開溶媒を十分量調製しておくことが重要です。途中で溶媒が足りなくなると、カラムが乾いて分離が台無しになります。

- **必要量の 1.5〜2 倍**を事前に調製する
- 混合溶媒はメスシリンダーで計量し、よく混合してから使う
- グラジエント用に複数の溶媒比率を準備する場合は、それぞれにラベルを貼っておく

溶媒の混合比は体積比で指定するのが一般的です。「ヘキサン / 酢酸エチル = 7:3」は「ヘキサン 70 mL + 酢酸エチル 30 mL」を意味します。精密な計量は不要ですが、メスシリンダーで大まかに測ってください。三角フラスコで目分量、というのは再現性が下がる原因になります。

溶媒を混合したあとは、十分に撹拌して均一にします。とくにヘキサンと酢酸エチルの混合では問題になりませんが、水と有機溶媒の組み合わせでは混和しない場合があるので注意してください。
