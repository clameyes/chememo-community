「再結晶したいけど、どの溶媒を使えばいいかわからない」——再結晶の成否は溶媒選択でほぼ決まると言っても過言ではありません。この記事では、溶媒を選ぶための考え方とテスト方法を解説します。対象読者は、有機化学実験で再結晶を行う学部生の方です。再結晶の手順全体は「[再結晶のやり方と基本手順](/posts/chememo/recrystallization)」を参照してください。

## 溶媒選択の基本条件

再結晶に適した溶媒は、次の 3 つの条件を満たすものです。

1. **高温で目的化合物をよく溶かす**（加熱すれば完全に溶ける）
2. **低温では溶解度が下がる**（冷却すると結晶が析出する）
3. **室温で目的化合物を溶かしすぎない**（室温で全部溶けてしまう溶媒はNG）

3 つ目の条件が見落とされやすいポイントです。「よく溶ける溶媒がいいだろう」と考えて極性の近い溶媒を選ぶと、室温でも溶けてしまい結晶が出てきません。目的化合物を「**高温では溶かすが、冷やすと追い出す**」溶媒を探すのがコツです。

化学の教科書でよく引用される原則に「Like dissolves like（似たものは似たものを溶かす）」があります。極性の高い化合物には極性溶媒、低い化合物には非極性溶媒が候補になりますが、あくまで出発点です。実際にテストしてみないとわかりません。

## 溶解テストの進め方

候補溶媒が決まったら、少量のスケールでテストします。試料 10〜20 mg 程度で十分です。

1. **室温で溶けるか確認** — 試験管に試料を取り、候補溶媒を 0.5 mL ほど加えて振る。室温で完全に溶ける場合、その溶媒は**溶かしすぎ**なので不適です
2. **加熱で溶けるか確認** — 室温で溶けなければ、湯煎（60〜80 ℃ 程度）で加温する。加熱で溶ければ候補として有望です。加熱しても溶けない場合は溶媒を変えます
3. **冷却で結晶が出るか確認** — 加熱で溶かした溶液を室温に放冷し、その後氷冷する。結晶が析出すれば合格です
4. **結晶の質を確認** — 析出した結晶の形状を見ます。針状や板状の結晶が理想的です。粉末状なら冷却が速すぎた可能性があり、オイル状なら別の溶媒を試します

経験上、1 つ目の溶媒で理想的な結果が出ることはあまり多くありません。3〜4 種類の溶媒を試すつもりでテストに臨むと、焦らず進められます。試料が貴重な場合でも、テストに使う量はごく少量なので、ここを惜しまない方が最終的な収率は良くなります。

## よく使われる単溶媒

下の表は、再結晶によく使われる溶媒の一覧です。化合物の極性に応じて候補を絞り込む出発点としてください。

| 溶媒 | 沸点（℃） | 極性 | 適する化合物の例 |
|-----|----------|-----|---------------|
| 水（$\ce{H2O}$） | 100 | 高 | 塩、糖、アミノ酸 |
| メタノール（$\ce{MeOH}$） | 65 | 高 | 極性の高い有機化合物 |
| エタノール（$\ce{EtOH}$） | 78 | 高 | 極性の高い有機化合物 |
| アセトン | 56 | 中 | 多くの有機化合物 |
| 酢酸エチル（$\ce{AcOEt}$） | 77 | 中 | エステル、ケトン類 |
| ジクロロメタン（$\ce{CH2Cl2}$） | 40 | 中 | ハロゲン化合物 |
| トルエン | 111 | 低 | 芳香族化合物 |
| ヘキサン（$\ce{C6H14}$） | 69 | 低 | 非極性化合物 |

ジクロロメタンは沸点が低く蒸発が速いため、再結晶の溶媒としては扱いにくい面があります。また、揮発性が高く皮膚からも吸収されるため、使用時は必ず手袋を着用し、ドラフトチャンバー内で作業してください。

迷ったときは、まず**エタノール**か**酢酸エチル**でテストするのが実用的です。この 2 つは極性の異なる幅広い化合物に対応でき、入手もしやすい溶媒です。

## 混合溶媒の使い方

単溶媒でうまくいかない場合は、**良溶媒と貧溶媒を組み合わせた混合溶媒**を使います。良溶媒は目的化合物をよく溶かす溶媒、貧溶媒はほとんど溶かさない溶媒です。

使い方の手順は以下のとおりです。

1. 目的化合物を良溶媒に加熱溶解する
2. 溶液が熱いうちに、貧溶媒を少しずつ加えていく
3. 溶液がわずかに白濁し始めたら貧溶媒の追加をやめる
4. 少量の良溶媒を加えて白濁を解消する
5. ゆっくり冷却する

よく使われる組み合わせを下にまとめます。

| 良溶媒 / 貧溶媒 | 用途 |
|----------------|------|
| エタノール / 水 | 汎用性が高い。極性化合物に |
| アセトン / 水 | 極性化合物。水溶性が中程度のものに |
| 酢酸エチル / ヘキサン | 中程度の極性をもつ化合物に |
| ジクロロメタン / ヘキサン | 低極性の化合物に |
| トルエン / ヘキサン | 芳香族化合物に |

混合溶媒の利点は、良溶媒と貧溶媒の比率を調整することで溶解度を細かくコントロールできることです。単溶媒では「室温でも溶けすぎる」か「加熱しても溶けない」の両極端になる化合物でも、混合比を変えることで最適な条件が見つかることがあります。

実験者の立場からの補足ですが、混合溶媒の再結晶で貧溶媒を一度に大量に加えると、急激に過飽和状態になりオイルとして析出することがあります。「少しずつ加えて様子を見る」が鉄則です。

## 化合物の極性から溶媒を選ぶ

化合物の極性がわかっている場合は、下の対応関係を参考にして候補溶媒を絞り込めます。

| 化合物の極性 | 化合物の例 | 候補溶媒 |
|------------|----------|---------|
| 高極性 | 塩、アミノ酸、糖 | 水、メタノール |
| 中〜高極性 | アミド、カルボン酸 | エタノール、アセトン |
| 中極性 | エステル、ケトン | 酢酸エチル、アセトン |
| 低〜中極性 | エーテル、ハロゲン化物 | ジクロロメタン、トルエン |
| 低極性 | 炭化水素、芳香族 | トルエン、ヘキサン |

TLC（薄層クロマトグラフィー）で化合物の Rf 値を調べておくと、極性の見当がつきやすくなります。Rf 値が高い（よく動く）展開溶媒は、その化合物にとって極性が近い溶媒であることを示しています。

ただし、この表はあくまで目安です。同じ官能基を持つ化合物でも分子量や置換基によって溶解度は大きく異なるため、最終的には溶解テストで確認するのが確実です。

溶媒を決めたあとの再結晶の手順は「[再結晶のやり方と基本手順](/posts/chememo/recrystallization)」を、結晶がうまく出ない場合は「[再結晶で結晶が出ないときの対処法](/posts/chememo/recrystallization-troubleshooting)」を参照してください。
